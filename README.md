# Rage-s-fastfetch-config

## 🇬🇧 English

This is my Fastfetch configuration — simple and hassle-free.
To use custom ASCII art, create a file at `~/.config/fastfetch/logo.txt` and paste your desired ASCII art into it.
If you don’t do anything, Fastfetch will automatically use your distribution’s default ASCII art.


## 🇦🇿 Azərbaycan

Bu mənim Fastfetch konfiqurasiyamdır — sadədir və baş ağrıtmır.
Custom ASCII art istifadə etmək üçün `~/.config/fastfetch/logo.txt` faylını yaradıb istədiyin ASCII art-ı ora əlavə etməlisən.
Əgər heç nə etməsən, Fastfetch avtomatik olaraq distributivinin default ASCII art-ını göstərəcək.
